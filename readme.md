# Setup
- copy htmc.js/htmc.min.js into your project.
- add it to your index

# Usage
see [example.html](static/example.html), and [foo.html](static/foo.html)

# Run Example
```sh
git clone https://github.com/hacke2man/htmc.git
cd htmc
pnpm install
pnpm run dev
```
